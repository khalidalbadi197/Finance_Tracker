# Finance Tracker

A personal finance tracking web app built with Python and Streamlit.

## Features
- Add and save expenses by category (Food, Transport, Bills, etc.)
- Edit or delete expenses
- Visualize spending with a bar chart by category
- Predict next month's total spending using linear regression

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
1. Clone the repository
2. Install dependencies:
   ```
   pip install streamlit pandas matplotlib scikit-learn
   ```
3. Run the app:
   ```
   streamlit run app.py
   ```
